﹞           BOOTLOADER Region             

每         PRELOADER, DSP_BL

每         Request: Update ALL or Skip All to update

每         Exception: Partial Update

 

﹞           Protected Invisible Region

每         __NODL_NVRAM, __NODL_PRO_INFO, __NODL_PROTECT_F, __NODL_PROTECT_S

每         Request: Protect data along with firmware upgrade process

每         Exception: data lost, __NODL_NVRAM is not found.

NOTE:

__NODL_NVRAM is required to exist, if not, error message will pop up and firmware upgrade will be stopped.

 

﹞           Normal Invisible Region

每         __NODL_PMT, __NODL_SECCFG＃

每         Request: Erased by PMT.

每         Exception: Wrongly erase

 

﹞           Normal Visible Region

每         UBOOT, BOOTIMG, ANDROID and so on.

每         Request: Erase old data on target by PMT, Download new data to target

每         Exception: Wrongly erase or download

﹞           Kept Visible Region

每         FAT, (Maybe USRDATA＃)

每         Request: Location not change among firmware upgrade

每         Location not change among firmware upgrade, could be skipped to update

每         Exception: Wrongly erase or download

 

﹞           Reserved Region

每         __RSV_NODL_OTP, __NODL_PMT, __NODL_BMTPOOL

每         Request: skipped, no action.

每         Exception: wrongly erased
